## :arrow_forward: For developers:

### :stop_button: Setup
```
1. Set an environment variable "MARTATORO_BOT_TOKEN" with the Discord bot token as value.
2. Make sure you have already installed node.
3. Check out this repository.
4. Run the "npm install" command.
```
### :stop_button: Compiling
```
1. Make sure you have already installed node.
2. Check out this repository.
3. Run the "npm run build" command.
4. Find the output in your dist folder.
```

### :stop_button: Scripts
| Command | Purpose |
|-|-|
| ```build``` | Compiles your code into your dist folder. |
| ```dev``` | Launches the Discord bot with ```nodemon``` for developers. |
| ```devStart``` | Launches the Discord bot without ```nodemon``` for developers. |
| ```format``` | Formats the code for developers. |
| ```start``` | Compiles the code and starts the bot for production. |
